
DSB Website - Final Bundle
==========================
Ce paquet contient les fichiers du site DSB.
Fichiers : index.html, style.css, script.js, assets/
Utilisez download_images.sh pour télécharger les images HD libres de droit.

Étapes :
1. Extraire le zip
2. (Optionnel) Lancer : bash download_images.sh
3. Uploader tout sur GitHub (index.html, style.css, script.js, assets/)
4. Visiter votre site via GitHub Pages.
